# 📊 Customer Shopping Behavior Analysis

A complete end-to-end data analytics project using **Python**, **MySQL**, **Power BI**, **business reporting** and **presentation** to uncover insights about customer demographics, spending behavior, product preferences, and loyalty patterns.

---

## 🚀 Project Overview

This project analyzes **3,900 customer transactions** to understand shopping patterns, identify revenue drivers, segment customers, and generate actionable recommendations for a retail business.

The workflow covers the **full analytics lifecycle**:
✔ Data cleaning & preparation (Python)
✔ SQL-based business analysis (MySQL)
✔ Interactive dashboard (Power BI)
✔ Insights + recommendations (Report)
✔ Project presentation (Slides)

---

## 🧩 Problem Statement

A retail company wants to understand how different customer groups shop, what drives purchases, and how to improve revenue, retention, and subscription adoption.
The project answers the central question:

**“How can we leverage customer shopping data to identify trends, improve engagement, and optimize marketing and product strategies?”**

---

## 📦 Deliverables

**1. Data Preparation & Modeling (Python)**

* Cleaned and transformed raw data
* Handled missing values (e.g., Review Ratings)
* Standardized features (snake_case)
* Created engineered features like `age_group` & `purchase_frequency_days`
* Uploaded cleaned data to MySQL for analysis

**2. Data Analysis (MySQL)**

* Revenue analysis by gender, age group, category
* High-spending discount users
* Top-rated products
* Shipping type comparison
* Subscription impact on spending
* Customer segmentation (New, Returning, Loyal)
* Repeat buyers & subscription behavior
* Category-level ranking & product insights

**3. Visualization & Insights (Power BI)**
Interactive dashboard showcasing:

* KPIs (total customers, avg. rating, avg. purchase amount)
* Revenue and sales by category
* Age-group breakdown
* Subscription behavior
* Shopping type filters (gender, shipping, category)

**4. Report & Presentation**

* Full project report with methodology, SQL results, and insights
* Presentation summarizing findings and recommendations

**5. GitHub Repository**
Includes all code & assets:
📁 Python scripts (cleaning, EDA, MySQL connection)
📁 SQL queries (business analysis)
📁 Dashboard file (PBIX)
📁 Project report (PDF)
📁 Presentation deck (PPT/PDF)

---

## 🔍 Key Insights (Summary)

Some major insights discovered (based on your report):

* **Males generate 2× more revenue** than females
* **Young Adults** contribute the highest revenue
* **Express shipping users spend more** than standard shipping users
* **Subscribers have higher lifetime value**, despite making fewer purchases
* **Accessories & Clothing** dominate both sales volume and revenue
* Several products (e.g., Hat, Sneakers, Sweater) are **highly discount-dependent**

---

## 💡 Business Recommendations

* **Boost Subscriptions** with exclusive perks and loyalty tiers
* **Review discount strategy** for heavily discounted items
* **Promote top-rated products** in targeted campaigns
* **Focus marketing on high-value age groups**
* **Reward repeat buyers** to push them into “Loyal” segment
* Highlight fast-shipping options to increase AOV
